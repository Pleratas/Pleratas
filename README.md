## Selected projects

### [PNA Alpha/Epsilon Dihedral Angle Statistics](https://github.com/Pleratas/angle-statistics)

A bioinformatics workflow for calculating and analysing selected PNA backbone dihedral angles from structural data.

**Technologies used:** Ada, Python, AWK, Makefile, LaTeX

Includes:

- an Ada command-line tool for extracting torsion angles;
- AWK/Python post-processing;
- summary statistics and plots;
- final analysis results;
- a written report.

---

### [PNA Torsion Angle Calculator](https://github.com/Pleratas/bioinformatics-assignment-2)

An Ada command-line program for extracting alpha and epsilon torsion angles from PDB and PDBx/mmCIF structural files.

**Technologies used:** Ada, Makefile, shell testing

The project focuses on parsing molecular structure files and calculating selected PNA backbone torsion angles.

---

### [Plane Atoms Generator](https://github.com/Pleratas/bioinformatics-assignment-1)

An Ada command-line program for generating atom coordinates on a 3D plane.

**Technologies used:** Ada, Makefile, shell testing

The program supports grid-based point generation and Halton-distributed point generation.

---

### [Perl Cryptography Tools](https://github.com/Pleratas/perl-cryptography-tools)

A small collection of Perl command-line programs related to basic cryptography.

**Technologies used:** Perl, Makefile, Git, SVN

Includes implementations of Caesar cipher and One-Time Pad related tools.
